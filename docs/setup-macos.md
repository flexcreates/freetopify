# macOS Setup

## Requirements
- Python 3.11+
- Homebrew

## Install
```bash
python3 freetopify.py install
```

## Start
```bash
python3 freetopify.py start
```

## Quick Checks
```bash
python3 freetopify.py doctor
python3 scripts/run_server.py --dry-run
```

## Optional CLI Helpers
```bash
python3 freetopify_media.py download
python3 freetopify_media.py organize
```

## Advanced Direct Scripts
```bash
./scripts/run_server_macos.sh
./scripts/ftsmdl.sh
./scripts/organize_music_library.sh
```
