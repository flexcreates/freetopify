# Windows Setup

## Requirements
- Python 3.11+
- PowerShell

## Install
```powershell
python freetopify.py install
```

## Start
```powershell
python freetopify.py start
```

## Quick Checks
```powershell
python freetopify.py doctor
python scripts/run_server.py --dry-run
```

## Optional CLI Helpers
```powershell
python freetopify_media.py download
python freetopify_media.py organize
```

## Advanced Direct Scripts
```powershell
.\scripts\run_server.ps1
.\scripts\ftsmdl.ps1
.\scripts\organize_music_library.ps1
```
